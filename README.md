# Puzzle Game

A simple and interactive sliding puzzle game built with HTML, CSS, and JavaScript. Players can move tiles to complete the puzzle and reveal the full image.

Features

Interactive puzzle: Click or drag tiles to solve the puzzle.

Responsive design: Works on desktop and mobile screens.

Visual effects: Smooth animations for tile movement.

Customizable background: Easily replace the puzzle image 

Technologies Used

HTML5 – Structure of the game.

CSS3 – Styling, gradients, animations, and responsive layout.

JavaScript (ES6) – Game logic, tile movement, and user interactions.
